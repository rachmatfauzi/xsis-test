## Project setup
```
composer install
```
OR
```
composer update
```
then

```
php artisan key:generate

php artisan jwt:secret

php artisan cache:clear

php artisan config:clear
```

### Database
```
execute xsis_test.sql in your database
```

### Compiles and hot-reloads for development
